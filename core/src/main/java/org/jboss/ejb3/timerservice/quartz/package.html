This package provides the EJB Timer Service to the EJB3 stack.

It replaces the org.jboss.ejb.txtimer package.

The Jboss EJB3 Timer Service package depends on:

  • EJB3 API
  • Quartz
  • JTA

Note that almost all exceptions are caught and rethrown as EJBExceptions, thus making sure that no Quartz API is exposed either compile time or run time.
