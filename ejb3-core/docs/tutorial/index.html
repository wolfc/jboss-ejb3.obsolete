EJB 3.0 Tutorial

This tutorial walks you through the EJB 3.0 features and how they deploy to JBoss. Please check the install guide for system requirements.

Installing

See Installing Guide

The Tutorial

You must set the JBOSS_HOME environment variable to run any of the tutorial examples. Also, you may want to edit local.properties in the base directory to override any build settings settings. For instance, you might want to set up jboss.home or jboss.server.config. By default the build files look in the 'default' configuration.
